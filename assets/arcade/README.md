# Team DC Arcade

Run from GRAV-MUD root:

```powershell
.\scripts\build-arcade-bundle.ps1
```

This copies itch/static builds into:

- `invaders/` — Emoji Invaders
- `galaga/` — Emoji Galaga
- `magick/` — Magick Emojis
- `flaplab/` — Magick Emojis FlapLab

Each folder needs an `index.html` at its root.
