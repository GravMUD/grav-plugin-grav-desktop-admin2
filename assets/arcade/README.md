# Team DC Arcade (optional assets)

Mambo Desktop expects each game as a static web build under this folder. Copy **your** build output here on the server (or dev machine), inside your normal Grav install:

```text
user/plugins/grav-desktop-admin2/assets/arcade/
  invaders/index.html
  galaga/index.html
  magick/index.html
  flaplab/index.html
```

| Folder | Game |
|--------|------|
| `invaders` | Emoji Invaders |
| `galaga` | Emoji Galaga |
| `magick` | Magick Emojis |
| `flaplab` | Magick Emojis: FlapLab |

These files are **not** committed to git (size). The plugin works without them; arcade apps simply need the folders before play.
