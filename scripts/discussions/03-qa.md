Stuck? Confused? Want to know if Grav Desktop plays nice with your stack? **Ask here.**

### Common starting points
**Requirements:** Grav 2.0+, admin2, api plugins enabled.

**Manual install:**
```bash
bin/gpm direct-install https://github.com/GravMUD/grav-plugin-grav-desktop-admin2/releases/download/v0.6.0/grav-plugin-grav-desktop-admin2.zip
```

**Docs / demos:** https://desktop.gravmud.site

**Arcade games:** copy builds into `assets/arcade/` (see repo README — not in the zip by default).

### Good things to include in your question
- Grav version (`bin/grav --version`)
- PHP version
- Whether you're on Admin2 (not legacy admin)
- Plugin settings (`show_delight_features`, `show_sticky_notes`, etc.)
- What you expected vs. what happened
- Screenshots if it's a visual thing

For **bugs**, you can also open an [issue](https://github.com/GravMUD/grav-plugin-grav-desktop-admin2/issues) — casual questions belong right here.

No dumb questions. Only unpainted flight decks.
