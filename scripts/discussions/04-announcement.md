## We shipped v0.6.0

**Site:** https://desktop.gravmud.site  
**Release:** https://github.com/GravMUD/grav-plugin-grav-desktop-admin2/releases/tag/v0.6.0

Mambo Desktop for Admin2 is a free web desktop inside Grav 2 Admin2 — icons, windows, taskbar, Start menu, operator tools, sticky notes, wallpaper strip, and Team DC Arcade.

### Phase A — Real
Site Vitals, Recent Pages, Spotify embed, JavaBean wallpaper sync

### Phase B — Operator
Log Tail, Maintenance toggle (`.upgrading`), API Smoke Test, GPM quick search

### Phase C — Delight
Sticky notes (per-user save), wallpaper preset strip, custom wallpaper upload

### Install
```bash
bin/gpm direct-install https://github.com/GravMUD/grav-plugin-grav-desktop-admin2/releases/download/v0.6.0/grav-plugin-grav-desktop-admin2.zip
```

Requires **Grav 2.0**, **admin2**, and **api**.
