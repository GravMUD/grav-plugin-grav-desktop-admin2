## Mambo Desktop for Admin2 v0.6.0

### Phase C — Delight
- Sticky notes on the desktop (draggable, per-user server save)
- Wallpaper preset strip above the taskbar (built-ins + JavaBean + custom)
- Custom wallpaper upload per admin user (WebP via API)

### Phase B — Operator
- Log Tail, Maintenance toggle, API Smoke Test, GPM Search

### Phase A — Real
- Site Vitals, Recent Pages, Spotify embed, JavaBean wallpapers

### Also
- Full window manager (drag, resize, maximize, session restore)
- Team DC Arcade (bundle games separately via `build-arcade-bundle.ps1`)
- Docs site: https://desktop.gravmud.site

**Install:** Download zip → `user/plugins/grav-desktop-admin2` → `bin/grav cache` → Admin2 sidebar → Mambo Desktop
